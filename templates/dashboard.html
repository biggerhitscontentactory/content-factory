📌 Pinterest Board Selector
Set PINTEREST_ACCESS_TOKEN in Railway Variables, then click Fetch Boards to load your boards. Map each product tier and category to a board, then choose which boards to repin into.
⚠️ PINTEREST_ACCESS_TOKEN not set — board posting and repinning will not work.
➕ Add Board Manually
Get your board ID from the Pinterest URL: open a board, look at the URL — pinterest.com/username/board-name/ — the numeric board ID is in the page source (right-click → View Source → search "board_id"). Or use the board URL slug itself as the ID (OneUp accepts both).
